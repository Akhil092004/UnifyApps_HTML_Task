# HTML Form Assignment

This project implements a user registration form using **only HTML and Vanilla JavaScript**, as per the given instructions.

## Features

- Form fields included:
  - Name (text)
  - Age (numbers only)
  - Gender (Male / Female / Other â€“ single selection)
  - Country (Dropdown: India, USA, Japan, China)
  - Technological Stack (Multiple selection: HTML, CSS, JS, React, React Native)
  - Address (Multi-line input)
  - Telephone (10-digit numeric input)
  - Email (Valid email format)

- Two buttons:
  - **Reset**: Clears the form and resets it to the initial state
  - **Submit**:
    - Validates that all fields are filled
    - Validates correct input types
    - Displays submitted data below the form in a table format

## Validation

- Empty field validation using JavaScript
- Input type validation using appropriate HTML5 input types
- Telephone number validation using pattern matching
- Email validation using `type="email"`

## Technologies Used

- HTML5
- Vanilla JavaScript (No external libraries)

## How to Run

1. Download or clone the repository
2. Open the HTML file in any modern web browser
3. Fill the form and click **Submit** to view the output

## Notes

- Labels and inputs are explicitly separated for clarity and accessibility
- No CSS frameworks or JavaScript libraries are used
- Designed to meet evaluation and readability standards
